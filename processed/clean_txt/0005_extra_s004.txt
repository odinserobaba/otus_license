Требования к компьютеру для установки УТМ 
Технические требования к оборудованию, соответствие которым обеспечит бесперебойную работу с ЕГАИС, для всех участников алкогольного рынка. 
Технические требования для работы с ЕГАИС описаны на портале egais.ru 
Процессор с тактовой частотой не менее 1.8 ГГц и поддержкой технологии SSE2; 
Оперативная память не менее 2 Гигабайт; 
Подключение к сети Интернет по выделенному каналу (минимальная скорость подключения — не менее 256 Кбит/сек, рекомендуемая скорость — от 1 Мбит/сек). Сетевой Ethernet контроллер, 100/ 1000 Mbps, разъём RJ45;

Свободное дисковое пространство не менее 50 Гигабайт; 
Аппаратный крипто-ключ с записанным на него сертификатом КЭП, обеспечивающий неизвлекаемость закрытых ключей - JaCarta SE или Рутокен ЭЦП 3.0; 
Программное обеспечение для работы с ЕГАИС:

Общесистемное ПО Java 8 и выше; 
Программное обеспечение ЕГАИС - УТМ последней доступной версии; 
ПО для работы с JaCarta SE/Рутокен ЭЦП 3.0. 
Лицензионная операционная система компании Microsoft:

для запроса RSA-сертификата - Windows Vista SP2/Server 2008 и выше (а также IE не ниже 9-ой версии), 
для работы УТМ - Windows 7 и выше 
Для работы УТМ должны быть: 
Открыт порт 443, доступны порты 8192, 8193, 8080 
Соблюдены штатные условия работы УТМ ( приложение Д ), в частности, на одном компьютере должен использоваться только один носитель (Джакарта или Рутокен) для работы с УТМ 
Открыт доступ ко всем адресам, прописанным в конфигурации УТМ: https://update.egais.ru/ , https://test.utm.egais.ru/ , https://update.egais.ru/monitor/file

Обновление УТМ 

Установка ПО ЕГАИС ФСРАР Популярные инструкции Программное обеспечение Ключевые носители Сертификаты Центр Загрузки Прочее ФГИС Меркурий Маркировка 

Обращаем внимание на наиболее частые причины неудачного обновления УТМ: 
использование антивирусного программного обеспечения, препятствующего установке обновления универсального транспортного модуля ЕГАИС (далее - УТМ); 
использование брандмауэра Windows, препятствующего установке обновления УТМ; 
службы УТМ запущены не от имени учетной записи Local system (Локальная система), которая обладает полными правами доступа для удаления/обновления/запуска служб УТМ. 
Для штатного обновления УТМ необходимо: 

отключить / настроить работу антивирусного программного обеспечения таким образом, чтобы оно не влияло на работу УТМ; 
отключить / настроить работу брэндмауэра таким образом, чтобы он не влиял на работу УТМ; 
обеспечить запуск УТМ от имени учетной записи Local system (Локальная система) с полными правами доступа 

Для обновления УТМ выполните следующие действия: 

В области уведомлений должно появиться сообщение о наличии новой версии УТМ. Значок агента при этом начнет мигать 

Нажмите на значок агента правой кнопкой мыши и выберите пункт Установить обновление 

Появится сообщение о процессе обновления. Значок агента начнет вращаться 

После обновления УТМ, проверьте корректность работы новой версии УТМ. 
Для проверки корректности работы УТМ необходимо открыть браузер Internet Explorer и перейти на страницу http://localhost:8080/ или нажать на значок УТМ правой кнопкой мыши, затем выбрать пункт Домашняя страница . 

Если при выполнении предыдущих действий УТМ не обновился. Выполните следующие действия: 

Нажмите на агент правой кнопкой мыши, выберите пункт Выйти 
Отключите антивирусное ПО 
Перезапустить службы УТМ по инструкции 
Запустите bat-файл агента ( C:\UTM\agent\bin\agent.bat ) от имени администратора и повторите попытку обновления 
Если обновление УТМ описанным способом не происходит, выполните следующие действия: 

Перед удалением УТМ, сохраните базу УТМ. 
Для сохранения зайдите в папку UTM (или ТТ) - transporter- transportDB. 
Скопируйте полностью папку transportDB на рабочий стол или на съемный носитель 

После этого следует удалить УТМ. Для удаления УТМ нужно зайти Пуск-Панель управления-Программы и Компоненты - ФСРАР УТМ, версия 2 
Удалите ФСРАР УТМ, версия 2 

Установите УТМ заново. Для этого воспользуйтесь инструкцией . 
Запустите УТМ по ярлыку UTM

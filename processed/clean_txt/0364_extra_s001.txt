Программа Навигатор УТМ более не поддерживается. 

Для работы с ЕГАИС рекомендуем использовать Master UTM . 

Навигатор УТМ позволяет выполнять простейшие запросы к УТМ: 

Справочников организаций; 
Сведений о контрагентах; 
Справочников алкогольной и спиртосодержащей продукции, спирта; 
Сведений штриховых кодов по типу, серии и номеру марки; 
Реестра необработанных ТТН; 

Запросов на переотправку входящих ТТН; 
Сведений об остатках продукции, числящихся в ЕГАИС за объектом (первый регистр); 
Сведений об остатках продукции, числящихся в ЕГАИС за торговым залом (второй регистр); 
Сведений о разделах «А» и «Б» справок к ТТН; 
Сведений о разделе А справки к ТТН (справка 1) 
Сведений о разделе Б спрвка к ТТН )справка 2) 
Сведений о движении продукции по форме Б 
Информирование об используемой xsd-схеме документов 
При выполнении запросов к УТМ необходимо указывать свой FSRAR_ID. Этот идентификатор указывается при добавлении УТМ в настройках приложения. 
Если ранее вы не указали этот параметр, выполните действия по инструкции . 
Запрос справочника организаций 

Для запроса сведений о той или иной организации, выполните следующее: 

Нажмите кнопку Запросы - Справочник организаций 
Укажите ИНН или FSRAR_ID организации 
Нажмите ОК 

УТМ отправит запрос по указанным вами параметрам. Отслеживать стадии обработки запроса вы можете во вкладке Журнал запросов 

Запрос сведений о контрагенте 
Для запроса сведений о контрагенте, выполните следующее: 

Нажмите кнопку Запросы - Сведения о контрагенте 

В открывшемся окне укажите FSRAR_ID контрагента и дату актуальности запрашиваемых сведений . Затем нажмите ОК 

После проверки введенных данных программа отправит ваш запрос сведений о контрагенте в УТМ 

Запрос справочника продукции 

Для запроса справочника продукции, выполните следующее: 

Нажмите кнопку Запросы - Справочник продукции 
Выберите тип запрашиваемой продукции (алкогольная, спиртосодержащая продукция или спирт) 
Укажите ИНН производителя/импортера или Код продукции 
Нажмите ОК 

УТМ отправит запрос по указанным вами параметрам. Отслеживать стадии обработки запроса вы можете во вкладке Журнал запросов 

Запрос сведений о штриховом коде 

Для запроса сведений о штриховом коде выполните следующее: 

Нажмите кнопку Запросы - Штрих-код с марками 

Заполните: номер и дату запроса, тип ФСМ/АМ, серию и номер марки. Нажмите ОК 

После проверки указанных данных, запрос будет отправлен 

Запрос реестра необработанных ТТН 

Для запроса реестра необработанных ТТН выполните следующее: 

Нажмите кнопку Запросы - Необработанные ТТН 

Затем нажмите ОК 

Повторный запрос ТТН 

Для повторного запроса ТТН, выполните следующее: 

Нажмите кнопку Запросы - Повтор ТТН 

Укажите идентификатор запрашиваемой ТТН в формате "TTN-0000000000", нажмите ОК 
УТМ отправит запрос по указанным вами параметрам. Отслеживать стадии обработки запроса вы можете во вкладке Журнал запросов 

Запрос сведений об остатках продукции 

Для запроса справочника продукции, выполните следующее: 

Нажмите кнопку Запросы. Выберите одну из кнопок: 
- Запросить сведения об остатках продукции 
- Запросить остатки товара в магазине 

В открывшемся окне нажмите ОК 

УТМ отправит запрос по указанным вами параметрам. Отслеживать стадии обработки запроса вы можете во вкладке Журнал запросов 

Запрос сведений из разделов А и Б к ТТН 

Для запроса справочника продукции, выполните следующее: 

Нажмите кнопку Запросы . Выберите одну из кнопок: 
- Запросить сведения о разделе А справки к ТТН (справки 1) 
- Запросить сведения о разделе Б справки к ТТН (справки 2) 
- Запросить движение по форме Б 

Укажите идентификатор запрашиваемого раздела в формате "FA-000000000000000" или "FB-000000000000000" 
Нажмите ОК 

УТМ отправит запрос по указанным вами параметрам. Отслеживать стадии обработки запроса вы можете во вкладке Журнал запросов 

Информирование об используемой версии xsd-схем документов 

Для запроса справочника продукции, выполните следующее: 

Нажмите кнопку Запросы . Нажмите кнопку Информация о версии xsd-схем 

Выберите версию xsd-схем, нажмите ОК

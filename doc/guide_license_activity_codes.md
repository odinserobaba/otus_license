# Справочник: виды лицензируемой деятельности и внутренние коды (ЕГАИС / классификация)

Документ служебный для поиска по RAG. Коды используются как краткие идентификаторы вида деятельности; формулировки слева — человекочитаемое описание.

| Код | Описание вида деятельности |
|-----|----------------------------|
| ПХП_ВИНО_ЗГУ | Производство вина с защищённым географическим указанием, с защищённым наименованием места происхождения |
| ПХП_ВИНО | Производство, хранение и поставки произведённого вина, игристого вина, плодовой алкогольной продукции, плодовых алкогольных напитков без добавления этилового спирта, виноградосодержащих напитков без добавления этилового спирта, плодовых сброженных материалов |
| ВРЗ | Временное разрешение на завершение полного цикла производства дистиллятов или право на завершение выдержки винодельческой продукции, а также право на хранение и поставку дистиллятов или винодельческой продукции |
| РПО | Розничная продажа алкогольной продукции при оказании услуг общественного питания |
| РПА | Розничная продажа алкогольной продукции |
| Т_ССНП | Перевозки нефасованной спиртосодержащей непищевой продукции с содержанием этилового спирта более 25 процентов объёма готовой продукции |
| Т_ССПП | Перевозки нефасованной спиртосодержащей пищевой продукции с содержанием этилового спирта более 25 процентов объёма готовой продукции |
| Т_ЭС | Перевозки этилового спирта |
| Х_ССНП | Хранение спиртосодержащей непищевой продукции |
| Х_ССПП | Хранение спиртосодержащей пищевой продукции |
| Х_АП | Хранение алкогольной продукции |
| Х_ЭС | Хранение этилового спирта |
| ЗХП_ССНП | Закупка, хранение и поставки спиртосодержащей непищевой продукции |
| ЗХП_ССПП | Закупка, хранение и поставки спиртосодержащей пищевой продукции |
| ЗХП_АП | Закупка, хранение и поставки алкогольной продукции |
| ПХП_ФАРМ | Производство этилового спирта для производства фармацевтической субстанции спирта этилового (этанола) |
| ПХП_ССНП | Производство, хранение и поставки спиртосодержащей непищевой продукции |
| ПХП_ССПП | Производство, хранение и поставки произведённой спиртосодержащей пищевой продукции |
| ПХПРП_СХП | Производство, хранение, поставки и розничная продажа произведённой сельскохозяйственными производителями винодельческой продукции |
| ПХП_СХП | Производство, хранение и поставки произведённой сельскохозяйственными производителями винодельческой продукции |
| ПХП_АП | Производство, хранение и поставки произведённой алкогольной продукции |
| ПХП_ЭС | Производство, хранение и поставки произведённого этилового спирта |

При запросе пользователя по коду (например Т_ЭС, ПХП_ВИНО) сопоставляйте код с описанием из этой таблицы и ищите в нормативных актах соответствующие лицензируемые виды деятельности (перевозка этилового спирта, розничная продажа, производство вина и т.д.).

Если в движении продукции по 1 регистру по справке Б есть красные строки, то выполните следующее: 

Откройте документ красного цвета. (Открыть документ можно прямо из движения продукции, щелкнув по нему левой кнопкой мыши 2 раза.) 

Если справки Б указаны вручную , то выполните следующие шаги: 

Найдите продукцию с искомой справкой Б по инструкции . 
Удалите в документе искомую справку Б, нажав Очистить. 
Нажмите ОК. 
Обновите БД через Сервис - Обновление БД - RecreateEntries.sql (обязательно создайте резервную копию перед обновлением БД по инструкции ). 

Если справки Б не указаны вручную , то обновите БД через Сервис - Обновление БД - RecreateEntries.sql (обязательно создайте резервную копию перед обновлением БД по инструкции ). 

Перейдите в раздел Движение продукции- Журнал документов 
Создайте новый документ Списание, нажмите кнопку Добавить строку 
Добавьте предварительно скопированное наименование продукции и выберите искомую продукцию. Посмотрите остаток по данной продукции. 

Нажмите на значок лупы в разделе Справка А данной продукции 

Посмотрите остаток продукции по партиям. 

- Если остаток по партиям равен остатку по 1 регистру, то значит, что остаток по партиям отображается корректный. 
- Если остаток по партиям больше или меньше остатка по 1 регистру, то тогда остаток по партиям отображается некорректный. 
Если при выборе партии продукции есть красные строки с минусами или строки с одинаковыми справками Б, то по таким строкам выполните все пункты данной инструкции с самого начала. 

- Если таких строк нет, а остаток по партиям отображается некорректный, то обновите БД через Сервис - Обновление БД - RecreateEntries.sql (обязательно создайте резервную копию перед обновлением БД по инструкции ).

Создание МЧД на портале ФНС для работы в ЕГАИС с подписью физлица

Для создания/регистрации машиночитаемых доверенностей (МЧД) на портале ФНС перейдите по ссылке m4d.nalog.gov.ru . 
На открывшемся сайте выполните следующие действия: 

Выберите раздел " МЧД " в левом меню и нажмите кнопку " Создать доверенность " 

Вам откроется форма создания доверенности. 
На первом шаге выберите тип доверенности - Единая форма доверенности , нажав на нее. 
После этого нажмите на кнопку ниже Ввод сведений . 

На втором шаге вам необходимо будет заполнить единую форму доверенности, состоящую из четырех пунктов сведений: 
• о доверенности 
• о доверителе 
• о представителе 
• о предоставляемых полномочиях 

Сведения о доверенности 

В сведениях о доверенности укажите внутренний номер доверенности и сроки действия доверенности. 
Затем нажмите " Готово ", и кнопку " Следующий раздел " для перехода на ввод сведений о доверителе. 

Сведения о доверителе 

Выберите категорию доверителя из списка и заполните все необходимые данные о доверителе (помечены красными звездочками). 
Обратите внимание - если включен пункт " Проверить сведения и автозаполнить поля ", то после после нажатия кнопки " Далее ", наименование организации или ИП автоматически введётся в поле из ЕГРЮЛ. 

Для доверителя нужно выбрать как минимум одного ЕИО (руководитель). Значения можно выбрать автоматически из списка (если ранее был выбран пункт автозаполнения, то сведения о руководителе частично берутся из ЕГРЮЛ), либо вводить полностью вручную (рекомендуется только если возникают ошибки при автозаполнении). 
Обратите внимание, что некоторые поля нужно заполнить вручную (дата рождения, СНИЛС). 

После корректного заполнения во втором разделе должно отображаться название организации, которая делает доверенность, а также руководитель организации. Для перехода в следующий раздел нажмите кнопку "Готово" и "Следующий раздел". 

Сведения о представителе 

В этом разделе указываются полные сведения о представителе. Аналогично доверителю - рекомендуется оставить переключатель автозаполнения. После ввода часть данных проходит проверку и автоматически заполняется (например, данные из паспорта по его серии и номеру). 

После заполнения данных о представителе нажмите кнопки "Готово" и "Следующий раздел" для заполнения данных о полномочиях. 

Сведения о предоставляемых полномочиях 

Для перехода к вводу полномочий в доверенности - нажмите "Далее", указав данные о передоверии. 

Для доверенности работы с ЕГАИС вам понадобится выбирать пункт "Монолитные полномочия". 
Сами полномочия выбираются из Единого классификатора. Для удобства вы можете написать вручную в строке поиска текст "Росалкогольрегулирование" и поставить галочки либо на самом разделе, либо поштучно на полномочиях. 
Необходимо выбрать все полномочия, начинающиеся с букв "RAR". 

После указания всех полномочий для работы с ЕГАИС из раздела Росалкогольрегулирование, нажимайте "Готово" и "Следующий раздел". 

Вы можете указать пароль к доверенности в следующем разделе или сразу переходить к результату по соответствующей кнопке 

Сохраните уникальный номер доверенности (идентификатор под заголовком "Доверенность"). 
Он понадобится для проверки статуса доверенности в реестре. 
После того, как вы убедились, что все данные верны, нажмите Сформировать XML, указав пункт согласия на обработку персональных данных. 

Файл МЧД сформирован. Но это еще не все. 

Далее сформированный файл необходимо подписать подписью руководителя организации/ИП - создать подписанный файл МЧД формата SIG или p7s. 

Для этого мы рекомендуем воспользоваться одним из способов подписания файла через сторонние программы: 
• ПО КриптоАРМ. Инструкция по подписанию файлов с помощью данного ПО находится здесь . 
• Приложение "Инструменты КриптоПро". 
Внимание! Предварительно должна быть установлена программа КриптоПро CSP версии 5! 
В приложении необходимо будет нажать "Создание подписи" - "Показать расширенные" - "Создать отсоединенную подпись", выбрать подпись из списка, нажать "Выбрать файл для подписи" - "Подписать" 

Таким образом, сформирована доверенность в формате xml и отсоединенная подпись в формате SIG или p7s. Прежде чем загружать их в УТМ для работы сотрудника в системе по доверенности, необходимо загрузить доверенность и подпись в распределенный реестр в соответствующем разделе. 
Сама доверенность обрабатывается в реестре довольно быстро. Проверить её статус можно в разделе "Проверить статус доверенности", здесь понадобится ввести уникальный номер, сохраненный ранее. 

Для добавления МЧД в ЕГАИС через УТМ воспользуйтесь инструкцией .

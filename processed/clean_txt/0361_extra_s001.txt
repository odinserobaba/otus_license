Программа Retail Declaration более не поддерживается. 

Для работы с ЕГАИС рекомендуем использовать Контур.Маркет . 

Создать документ возврата поставщику можно двумя способами: автоматически из документа поступления (ТТН) и создать документ возврата полностью вручную. 

Обратите внимание , перед формированием документа Возврат поставщику проверьте настройки получателя данного документа. Для этого откройте раздел Справочники/Организации, дважды нажмите на подразделение получателя продукции и укажите Версия 3 в строке Версия xsd-схем. 

Для создания документа возврата поставщику выполните следующее: 
Создание документа возврата из ТТН 

Нажмите Движение продукции , выберите пункт Журнал документов 
Найдите нужный документ поступления от поставщика. Нажмите на него правой кнопкой мыши, выберите пункт Ввести на основании , затем нажмите Возврат поставщику 

Укажите все необходимые данные. Затем нажмите ОК 

В списке документов появится документ Возврат поставщику 
Для его отправки, выберите документ и нажмите на кнопку Отправить документ через УТМ (или нажмите правой кнопкой мыши на документ и выберите пункт Отправить ) 

Через 3-5 минут нажмите Запросить входящие документы через УТМ , чтобы проверить, зарегистрировался ли документ в ЕГАИС. Дождитесь, когда документ зарегистрируется в ЕГАИС (в этом случае статус документа изменится на «ТТН проведена»). 

Создание документа возврата вручную 
Для создания документа о возврате продукции поставщику выполните следующие действия: 

Нажмите Движение продукции , выберите пункт Журнал документов . Затем нажмите Добавить (или нажмите правой кнопкой мыши на пустое поле и выберите Создать) 

В открывшемся окне выберите тип создаваемого документа Возврат поставщику , затем нажмите ОК 

В открывшемся окне заполните все данные (номер документа, Контрагент, Подразделение и др.) в шапке документа. 

Для алкогольной продукции, за исключением немаркированной АП, добавлять позиции вручную из списка не нужно, информация о продукции заполнится автоматически при считывании штрих-кода с АМ/ФСМ. Для этого отсканируйте длинный штрих-код акцизной марки или федеральной специальной марки. 

Количество продукции = количеству отсканированных марок. 

Если однозначно определить продукцию нельзя - имеется несколько карточек продукции с отсканированным штриховым кодом или остаток продукции не положителен, то появится окно выбора продукции: 

Для добавления немаркированной продукции нажмите Добавить- Добавить из остатков. 

Выберите нужную продукцию из списка, укажите количество для возврата. Нажмите ОК 

Укажите все необходимые данные. Затем нажмите ОК 

В списке документов появится документ Возврат поставщику. 
Для его отправки, выберите документ и нажмите на кнопку Отправить документ через УТМ (или нажмите правой кнопкой мыши на документ и выберите пункт Отправить ) 

Через 3-5 минут нажмите Запросить входящие документы через УТМ , чтобы проверить, зарегистрировался ли документ в ЕГАИС. Дождитесь, когда документ зарегистрируется в ЕГАИС (в этом случае статус документа изменится на «ТТН проведена»).
